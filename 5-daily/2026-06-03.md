## 📝 学习笔记
今天跟 vibecoding 老师学了如何搭建个人知识库。

**关键收获：**
- 知识库的核心不是「存」，而是「用」
- Markdown 格式的笔记兼容性最好
- PARA 方法：Projects / Areas / Resources / Archives
- 模板化可以降低记录的门槛

## 💡 新发现
- 用 VS Code 打开整个知识库文件夹，编辑体验非常好
- `#标签` 可以让笔记之间产生关联
- 定期回顾比大量记录更重要

## ❓ 疑问
- 问：笔记太多了怎么找？
- 答：可以用 VS Code 的全局搜索 (Ctrl+Shift+F)，或者用 Obsidian 的可视化图谱
