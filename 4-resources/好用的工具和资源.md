## 💬 好用的 AI 提示词

### 学习新技术时
```
我想学 [技术名称]，我目前的基础是 [你的水平]。
请帮我制定一个学习路线，包含：
1. 先学什么后学什么
2. 每个阶段的练习项目
3. 常见坑点和注意事项
```

### 调试代码时
```
我的代码报了这个错：[粘贴错误信息]
这是我的代码：[粘贴代码]
我的目标是：[描述你想要的]
请帮我分析原因并修复。
```

### 审查代码时
```
请帮我审查这段代码：
[粘贴代码]
关注：代码结构、潜在 bug、性能问题、可读性
```
