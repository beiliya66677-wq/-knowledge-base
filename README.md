# 🧠 我的个人知识库

> 搭建日期：2026-06-03
> 一个用 Markdown 驱动的第二大脑，记录学习、代码和想法。

---

## 📂 目录结构

| 文件夹 | 用途 | 何时使用 |
|--------|------|----------|
| `0-inbox/` | 📥 收件箱 | 快速记录想法、临时笔记，等待日后整理 |
| `1-learning/` | 📖 学习笔记 | 系统学习某门技术、语言、框架时的笔记 |
| `2-projects/` | 🚀 项目记录 | 做过的项目、遇到的问题和解决方案 |
| `3-code-snippets/` | 🧩 代码片段 | 可复用的代码块、常用命令、配置模板 |
| `4-resources/` | 🔗 资源收藏 | 好文章、视频、工具的链接和摘要 |
| `5-daily/` | 📅 每日记录 | 每天的学习日志、遇到的问题和收获 |
| `templates/` | 📋 模板 | 新建笔记时复制使用的模板 |

---

## 🚀 快速开始

### 新建一个学习笔记
```bash
cp templates/concept.md "1-learning/vibecoding/我的新笔记.md"
```

### 快速记录一个想法
```bash
echo "# 想法" >> "0-inbox/$(date +%Y%m%d-%H%M).md"
```

### 写今天的日志
```bash
cp templates/daily-note.md "5-daily/$(date +%Y-%m-%d).md"
```

## 🏷️ 标签系统

在笔记中使用 `#标签` 来关联内容：

- `#vibecoding` — AI 辅助编程
- `#prompt` — 提示词技巧
- `#debug` — 调试记录
- `#tool` — 工具使用
- `#concept` — 概念理解
- `#todo` — 待办事项
